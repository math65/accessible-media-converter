Convertir des vidéos

Pour les sorties vidéo, l'application sépare clairement le conteneur, les réglages vidéo, les réglages audio et la gestion des pistes. C'est la logique à garder en tête.

Dans cette page

  • Choisir un conteneur
  • Régler la vidéo
  • Régler l'audio
  • Gérer les pistes

Choisir un conteneur

Les sorties conteneur vidéo principales sont MP4 et MKV. Vous pouvez également convertir une vidéo vers un format audio.

  • MP4 Format largement compatible. Les codecs audio utiles sont volontairement limités à AAC et MP3.
  • MKV Format plus souple. Les codecs audio utiles incluent AAC, MP3, Opus et FLAC.

Régler la vidéo

La section vidéo propose d'abord un preset, puis des réglages avancés si vous voulez reprendre la main.

Preset vidéo et Encoder Preset ne désignent pas la même chose. Le Preset vidéo est un profil global qui règle d'un coup la qualité, le preset d'encodage, le profil H.264 et le pixel format. Le Encoder Preset agit seulement sur la vitesse d'encodage et l'efficacité de compression du moteur x264.
Preset Usage
Compatible Choix sûr pour une lecture simple et robuste.
Balanced Compromis entre qualité, vitesse et taille.
High Quality Plus de qualité, encodage plus lent.
Small File Réduit davantage la taille du fichier.
Fast Encode Réduit le temps de traitement.

Les réglages avancés vidéo portent sur :

  • CRF
  • x264 preset
  • profil H.264
  • pixel format

En pratique, si vous voulez un réglage simple, choisissez d'abord un Preset vidéo. Si vous modifiez ensuite le Encoder Preset, le CRF, le profil ou le pixel format, vous sortez du preset global et passez sur un réglage plus personnalisé.

Régler l'audio dans une sortie vidéo

La section audio reste distincte de la section vidéo. Vous choisissez le codec audio, puis les réglages de bitrate, qualité, canaux et normalisation si nécessaire.

Pour une sortie vidéo, la logique est toujours : conteneur d'abord, puis réglages vidéo, puis réglages audio.

Gérer les pistes

Pour ouvrir les actions de pistes, faites un clic droit sur un fichier vidéo dans la liste. Le menu proposé dépend du format de sortie actuellement sélectionné.

  • Manage Tracks... Disponible pour une sortie vidéo MP4 ou MKV.
  • Choose Audio Track... Disponible pour une extraction audio depuis une vidéo avec plusieurs pistes audio.

Le gestionnaire de pistes permet ensuite notamment de :

  • conserver ou retirer des pistes vidéo, audio et sous-titres ;
  • réordonner les pistes ;
  • modifier le titre et la langue ;
  • ajuster certaines dispositions comme default ou forced.

Pour une conversion vidéo vers audio, l'application n'utilise qu'une seule piste audio. Si plusieurs pistes existent, un sélecteur dédié permet de choisir laquelle extraire.

Si aucune action de piste n'apparaît au clic droit, vérifiez d'abord le format de sortie choisi et le type réel du fichier sélectionné.

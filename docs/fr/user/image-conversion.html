Convertir des images

L'onglet Image convertit une image vers un autre format, avec un redimensionnement optionnel. Les images ajoutées à l'application sont automatiquement routées vers cet onglet.

Dans cette page

  • Choisir un format de sortie
  • Régler la qualité
  • Redimensionner
  • Traitement par lot

Choisir un format de sortie

Les formats image pris en charge sont :

  • JPEG
  • PNG (sans perte)
  • WebP
  • TIFF (sans perte)
  • BMP (non compressé)

Le format choisi détermine les réglages disponibles : un format avec perte comme JPEG propose une qualité, un format sans perte comme PNG propose un niveau de compression.

Régler la qualité

  • Qualité (JPEG, WebP) De 1 à 100. Plus la valeur est élevée, meilleure est la qualité et plus le fichier est gros.
  • Compression (PNG) De 0 (rapide) à 9 (fichier le plus petit), sans perte de qualité.
  • Sans perte (WebP) Une case à cocher active le mode WebP sans perte.
  • Compression (TIFF) LZW, Deflate, PackBits ou aucune.

Redimensionner

Un réglage de redimensionnement permet de réduire l'image vers une résolution cible (par exemple 4K, Full HD, HD). Le ratio est conservé automatiquement, donc l'image n'est jamais déformée.

Traitement par lot

Comme pour l'audio et la vidéo, l'application peut convertir plusieurs images en parallèle. Le nombre maximal de tâches simultanées se règle dans les préférences.

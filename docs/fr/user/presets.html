Presets d'encodage

Un preset est un ensemble de réglages de conversion enregistré, que vous réutilisez en deux clics. Préparez une fois un profil « Podcast » ou « Livre audio », puis appliquez-le quand vous en avez besoin au lieu de régler chaque option à la main.

Ouvrir la fenêtre des presets

La gestion des presets s'ouvre depuis trois endroits :

  • Le bouton Presets…, à côté du bouton Réglages / Qualité.
  • Un bouton Presets… sur l'écran d'accueil, disponible même sans fichier chargé — pratique pour importer un preset avant d'ajouter quoi que ce soit.
  • La liste des fichiers : clic droit (ou touche Menu / Maj+F10) puis Gérer les presets…

Les presets sont regroupés par onglet actif : les presets audio apparaissent sur l'onglet Audio, les presets vidéo sur l'onglet Vidéo et les presets image sur l'onglet Image.

Ce que contient un preset

  • Format de sortie Par exemple MP3, AAC, M4B, MP4, JPEG…
  • Réglages d'encodage Qualité, débit, mono/stéréo, et toutes les autres options de la fenêtre Réglages / Qualité.
  • Destination de sortie Le dossier de sortie et l'option « conserver l'arborescence des sous-dossiers d'origine ».
  • Modèle de métadonnées Un ensemble de tags communs (genre, artiste, album…) appliqués à vos fichiers. Les images n'ont pas de modèle de métadonnées.

Enregistrer, appliquer et gérer les presets

La fenêtre des presets liste vos presets par nom et propose :

  • Enregistrer les réglages actuels dans un preset Capture le format, les réglages d'encodage et la destination de sortie actuels sous un nom que vous choisissez.
  • Appliquer Restaure le preset sélectionné en une étape. Vous pouvez aussi double-cliquer un preset dans la liste pour l'appliquer.
  • Remplacer par les réglages actuels Écrase le preset sélectionné avec vos réglages actuels, en conservant son nom et son modèle de métadonnées.
  • Modifier les métadonnées Modifie le modèle de tags communs du preset (indisponible pour les images).
  • Renommer et Supprimer Gèrent la liste. Les noms ne tiennent pas compte de la casse.

Appliquer aux fichiers sélectionnés

Appliquer un preset définit toujours le format de sortie, les réglages d'encodage et la destination de sortie de manière globale — c'est le but d'un preset. Le modèle de métadonnées, lui, se comporte différemment selon la façon dont vous avez ouvert la fenêtre :

  • Depuis le bouton Presets…, les métadonnées sont appliquées à tous les fichiers chargés.
  • Depuis le menu clic droit de la liste, les métadonnées ne sont appliquées qu'aux fichiers sélectionnés.

Importer et exporter

Exporter enregistre vos presets dans un fichier que vous pouvez sauvegarder ou copier sur un autre ordinateur ; Importer les relit. À l'import, si un nom existe déjà, on vous demande s'il faut remplacer, conserver les deux, ou ignorer.

À l'export, vous pouvez choisir ce qui est inclus. Le dossier de sortie et le modèle de métadonnées peuvent être laissés de côté — le dossier de sortie, en particulier, est rarement pertinent d'un ordinateur à l'autre. Le format et les réglages d'encodage sont toujours inclus.

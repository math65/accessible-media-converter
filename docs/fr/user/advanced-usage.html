Réglages vidéo avancés

Pour les sorties MP4 et MKV, la vidéo peut être copiée ou réencodée. En mode réencodage, vous disposez d’un preset principal, puis de réglages avancés.

Le Preset vidéo est un raccourci global. Il applique ensemble plusieurs valeurs cohérentes, notamment le CRF, le Encoder Preset, le profil H.264 et le pixel format. Le Encoder Preset, lui, ne règle qu'un seul point : l'arbitrage entre vitesse d'encodage et efficacité de compression.

  • CRF pour piloter la qualité cible ;
  • x264 preset pour arbitrer vitesse contre compression ;
  • profil H.264 pour la compatibilité ;
  • pixel format pour le rendu et la lecture.
Si vous changez manuellement le Encoder Preset ou un autre réglage avancé, vous êtes en train de personnaliser le preset vidéo choisi au départ.

Réglages audio avancés

Dans une sortie vidéo, les réglages audio restent séparés des réglages vidéo. Vous pouvez choisir un codec compatible avec le conteneur, puis ajuster bitrate, qualité, canaux, fréquence et normalisation selon le codec choisi.

  • MP4 : AAC ou MP3
  • MKV : AAC, MP3, Opus ou FLAC

Gestion des pistes

Le gestionnaire de pistes permet de choisir quelles pistes vidéo, audio et sous-titres garder, de modifier leur ordre, puis d’ajuster certaines métadonnées et dispositions.

Dans la pratique, ces actions s'ouvrent par clic droit sur un fichier vidéo dans la liste. Manage Tracks... est proposé pour une sortie MP4 ou MKV, tandis que Choose Audio Track... apparaît pour une extraction audio avec plusieurs pistes.

Pour une conversion vidéo vers audio, l’application n’extrait qu’une seule piste audio.

Batch et performances

L’application peut lancer plusieurs conversions en parallèle. Les préférences permettent de définir :

  • le nombre maximal de conversions simultanées ;
  • la politique de conflit sur les fichiers de sortie ;
  • le nombre de threads FFmpeg par conversion.

Debug, support et mises à jour

Si un problème doit être signalé, vous pouvez activer le debug, reproduire le bug, puis ouvrir le dialogue de support intégré. Le formulaire transmet un rapport propre, sans exposer les chemins locaux complets.

Le dialogue de support peut aussi afficher les détails techniques inclus et proposer une copie manuelle du rapport si l'envoi automatique échoue.

Les mises à jour peuvent aussi être vérifiées manuellement depuis le menu Aide, ou automatiquement au lancement si l’option est activée.

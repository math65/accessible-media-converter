Usage avancé

Fusion de fichiers

Vous pouvez fusionner deux fichiers ou plus en une seule sortie (concat FFmpeg). Lorsque la cible est le format M4B (livre audio), la fusion génère un chapitre par fichier source. Convertir un seul fichier déjà chapitré vers M4B préserve ses chapitres, et le .m4b est aussi accepté en entrée. Par défaut stéréo 128 kb/s, mais tout est réglable (mono / 64 kb/s convient bien à la voix). Sélectionnez au moins deux fichiers (Ctrl+clic, Maj+clic ou Ctrl+A) puis lancez la fusion.

Édition des métadonnées

Clic droit sur un fichier audio ou vidéo, ou plusieurs (clic droit, ou touche Menu/Applications, ou Maj+F10, sur la ligne sélectionnée), puis Modifier les métadonnées… : tags standard (titre, artiste, album, artiste de l'album, compositeur, année, piste, disque, genre, commentaire) et pochette (remplacer JPEG/PNG, retirer, conserver). Cible : appliquer pendant la conversion ou re-taguer l'original maintenant. Les champs renseignés peuvent être appliqués à tout un lot.

Pour une vidéo, un sélecteur Type de contenu (Film / Série TV / Autre) adapte les champs (Film : synopsis ; Série : série, saison, épisode, titre d'épisode). Les fichiers audio proposent aussi Regroupement, Copyright et un champ Paroles multiligne.

Auto-détection depuis le nom de fichier

L'éditeur peut remplir certains champs depuis le nom de fichier, proposé via une case à cocher : pour une vidéo il lit la saison et l'épisode (motif SxxExx), et pour un fichier audio il lit le numéro de piste. Laissez la case décochée pour ne pas toucher aux champs.

Réglages vidéo avancés

Pour les sorties MP4 et MKV, la vidéo peut être copiée ou réencodée. En mode réencodage, vous disposez d’un preset principal, puis de réglages avancés.

Le Preset vidéo est un raccourci global. Il applique ensemble plusieurs valeurs cohérentes, notamment le CRF, le Encoder Preset, le profil H.264 et le pixel format. Le Encoder Preset, lui, ne règle qu'un seul point : l'arbitrage entre vitesse d'encodage et efficacité de compression.

  • CRF pour piloter la qualité cible ;
  • x264 preset pour arbitrer vitesse contre compression ;
  • profil H.264 pour la compatibilité ;
  • pixel format pour le rendu et la lecture.
Si vous changez manuellement le Encoder Preset ou un autre réglage avancé, vous êtes en train de personnaliser le preset vidéo choisi au départ.

Réglages audio avancés

Audio dans une sortie vidéo. Dans une sortie vidéo, les réglages audio restent séparés des réglages vidéo. Vous pouvez choisir un codec compatible avec le conteneur, puis ajuster bitrate, qualité, canaux, fréquence et normalisation selon le codec choisi.

  • MP4 : AAC ou MP3
  • MKV : AAC, MP3, Opus ou FLAC

Audio comme format de sortie principal. Lorsque la sortie est un format audio, vous pouvez choisir parmi MP3, AAC/M4A, WAV, FLAC, ALAC, OGG/Vorbis, WMA et M4B.

  • Copie de flux pour remuxer sans réencoder quand le codec source correspond ;
  • mode de débit CBR ou VBR selon le codec ;
  • bitrate, fréquence et canaux (par exemple mono / stéréo) ;
  • normalisation vers une cible de -16 LUFS ;
  • niveau de compression et profondeur de bits FLAC pour une sortie sans perte.

Gestion des pistes

Le gestionnaire de pistes permet de choisir quelles pistes vidéo, audio et sous-titres garder, de modifier leur ordre, puis d’ajuster certaines métadonnées et dispositions.

Dans la pratique, ces actions s'ouvrent par clic droit sur un fichier vidéo dans la liste (clic droit, ou touche Menu/Applications, ou Maj+F10, sur la ligne sélectionnée). Gérer les pistes... est proposé pour une sortie MP4 ou MKV, tandis que Choisir la piste audio... apparaît pour une extraction audio avec plusieurs pistes.

Pour une conversion vidéo vers audio, l’application n’extrait qu’une seule piste audio.

Batch et performances

L’application peut lancer plusieurs conversions en parallèle. Les préférences permettent de définir :

  • le nombre maximal de conversions simultanées ;
  • la politique de conflit sur les fichiers de sortie ;
  • le nombre de threads FFmpeg par conversion.

Debug, support et mises à jour

Si un problème doit être signalé, vous pouvez activer le debug, reproduire le bug, puis ouvrir le dialogue de support intégré. Le formulaire transmet un rapport propre, sans exposer les chemins locaux complets.

Le dialogue de support peut aussi afficher les détails techniques inclus et proposer une copie manuelle du rapport si l'envoi automatique échoue. Signaler un problème exige une adresse e-mail valide, et le rapport est bloqué si votre version est obsolète (une vérification de mise à jour est effectuée d'abord, car le bug est peut-être déjà corrigé).

Les mises à jour peuvent aussi être vérifiées manuellement depuis le menu Aide, ou automatiquement au lancement si l’option est activée. Avant son exécution, l'installeur téléchargé est vérifié pour son intégrité (taille du fichier et empreinte SHA-256). Après une mise à jour silencieuse, l'application se relance automatiquement.

Autres préférences à noter :

  • Nommage des chapitres M4B : titre sinon numéro localisé / titre sinon nom de fichier / toujours numéroté ;
  • Conserver les métadonnées d'origine : garder les tags source pendant la conversion.

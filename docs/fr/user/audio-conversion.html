Convertir des fichiers audio

L'onglet Audio est pensé pour rester simple : vous choisissez un format de sortie, puis vous ajustez la qualité, le bitrate, les canaux et la normalisation si besoin.

Dans cette page

  • Choisir un format de sortie
  • Régler la qualité
  • Normalisation streaming
  • Traitement par lot

Choisir un format de sortie

Les formats audio courants incluent :

  • MP3
  • AAC / M4A
  • WAV
  • FLAC
  • ALAC / M4A
  • OGG
  • WMA

Le format choisi détermine ensuite les réglages disponibles. Un format sans perte comme FLAC n'a pas la même logique qu'un format avec perte comme MP3 ou AAC.

Régler la qualité

  • Copy Stream Copie le flux audio existant sans réencodage, quand le conteneur ou le format le permet.
  • Convert Réencode l'audio avec les paramètres choisis.
  • CBR Utilise un bitrate constant.
  • VBR Utilise une qualité variable pour optimiser le rapport qualité / taille.
  • Sample rate, canaux, bit depth Permettent d'affiner la sortie selon le codec choisi.

Normalisation streaming

Une option permet de normaliser l'audio pour le streaming autour de -16 LUFS. Elle s'applique seulement quand l'audio est réencodé.

Si vous laissez l'audio en Copy Stream, aucune normalisation n'est appliquée.

Traitement par lot

L'application peut lancer plusieurs conversions en parallèle sur l'onglet courant. Le nombre maximal de tâches simultanées se règle dans les préférences.

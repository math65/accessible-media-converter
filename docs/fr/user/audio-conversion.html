Convertir des fichiers audio

L'onglet Audio est pensé pour rester simple : vous choisissez un format de sortie, puis vous ajustez la qualité, le bitrate, les canaux et la normalisation si besoin.

Dans cette page

  • Choisir un format de sortie
  • Livre audio M4B (chapitres)
  • Régler la qualité
  • Éditer les métadonnées audio
  • Normalisation streaming
  • Traitement par lot

Choisir un format de sortie

Les formats audio courants incluent :

  • MP3
  • AAC / M4A
  • WAV
  • FLAC
  • ALAC / M4A
  • OGG
  • WMA
  • M4B (livre audio avec chapitres)

Le format choisi détermine ensuite les réglages disponibles. Un format sans perte comme FLAC n'a pas la même logique qu'un format avec perte comme MP3 ou AAC.

Livre audio M4B (chapitres)

Le format de sortie M4B produit un livre audio avec chapitres. Le M4B est aussi accepté en entrée.

  • Fusionner plusieurs fichiers vers un M4B crée un chapitre par fichier.
  • Convertir un seul fichier déjà chapitré préserve ses chapitres.
  • Par défaut stéréo à 128 kb/s ; entièrement réglable (par ex. mono / 64 kb/s pour la voix).

Pour assembler plusieurs fichiers en un livre audio chapitré, utilisez la fusion (sélectionnez au moins deux fichiers) avec une sortie M4B. Le nommage des chapitres se règle dans les préférences.

Régler la qualité

  • Copy Stream Copie le flux audio existant sans réencodage, quand le conteneur ou le format le permet.
  • Convert Réencode l'audio avec les paramètres choisis.
  • CBR Utilise un bitrate constant.
  • VBR Utilise une qualité variable pour optimiser le rapport qualité / taille.
  • Sample rate, canaux, bit depth Permettent d'affiner la sortie selon le codec choisi.
  • Profondeur de bits et compression FLAC Spécifiques aux formats sans perte (comme WAV, FLAC, ALAC) ; elles ne s'appliquent pas aux formats avec perte comme MP3 ou AAC.

Éditer les métadonnées audio

Clic droit sur un ou plusieurs fichiers audio (ou touche Menu/Applications, ou Maj+F10, sur la ligne sélectionnée), puis Modifier les métadonnées…. Modifiez les tags : titre, artiste, album, artiste de l'album, compositeur, année, piste, disque, genre, commentaire, ainsi que Regroupement, Copyright et Paroles (multiligne). La pochette peut être remplacée (JPEG/PNG), retirée ou conservée.

Une case Détecter le numéro de piste depuis le nom de fichier remplit le numéro de piste. Choisissez la cible : appliquer pendant la conversion ou re-taguer l'original maintenant. L'édition par lot applique les champs renseignés à tous les fichiers sélectionnés.

Normalisation streaming

Une option permet de normaliser l'audio pour le streaming autour de -16 LUFS. Elle s'applique seulement quand l'audio est réencodé.

Si vous laissez l'audio en Copy Stream, aucune normalisation n'est appliquée.

Traitement par lot

L'application peut lancer plusieurs conversions en parallèle sur l'onglet courant. Le nombre maximal de tâches simultanées se règle dans les préférences.

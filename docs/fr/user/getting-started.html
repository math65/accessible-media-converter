Prendre en main Accessible Media Converter

Utilisez cette page si vous découvrez l'application. Elle couvre l'ouverture, l'ajout de médias, les formats acceptés et la logique générale des onglets (Audio, Vidéo et Image).

Démarrage rapide

  1. Ouvrez l'application.
  2. Ajoutez un ou plusieurs fichiers médias.
  3. Choisissez le format de sortie dans la liste de l'onglet actif.
  4. Ouvrez les réglages si vous voulez ajuster la qualité.
  5. Lancez la conversion avec Démarrer la conversion.
L'onglet actif décide du type de conversion : audio vers audio dans l'onglet Audio, vidéo vers vidéo ou vidéo vers audio dans l'onglet Vidéo.

Ajouter des fichiers ou un dossier

Trois méthodes principales existent :

  • Ctrl+O ou Fichier > Ajouter des fichiers...
  • Ctrl+V pour coller des fichiers copiés depuis l'explorateur Windows
  • Fichier > Ajouter un dossier... pour importer un dossier entier

Le collage et l'import de dossier appliquent le même filtrage que l'ouverture classique. Les fichiers non média sont ignorés.

L'application ne prend pas en charge le glisser-déposer. Utilisez l'ouverture, l'import de dossier ou le collage Ctrl+V.

Formats pris en charge

Type Extensions principales
Audio .mp3, .mp2, .wav, .flac, .aac, .ogg, .opus, .wma, .m4a, .m4b, .aiff, .aif, .ac3, .eac3, .dts, .mka, .amr
Vidéo .mp4, .mkv, .avi, .mov, .wmv, .webm, .ts, .m2ts, .mts, .mpg, .mpeg, .vob, .m4v, .3gp, .3g2, .flv, .ogv
Image .jpg, .jpeg, .png, .webp, .avif, .tiff, .bmp, .heic, .heif
Un fichier texte, un document bureautique ou tout autre format non listé ne doit pas être importé comme média.

Comprendre l'interface

  • Onglet Audio Utilisez-le pour convertir un fichier audio vers un autre format audio.
  • Onglet Vidéo Utilisez-le pour convertir une vidéo vers MP4 ou MKV, ou pour extraire son audio.
  • Onglet Image Utilisez-le pour convertir une image en JPEG, PNG, WebP, TIFF ou BMP, avec un redimensionnement optionnel.
  • Liste centrale Chaque ligne affiche le nom du fichier, quelques détails et le statut de traitement.
  • Barre de statut et progression Elles montrent l'état global du batch en cours.

Où trouver les actions avancées

Certaines actions n'apparaissent pas comme de gros boutons dans la fenêtre principale. Pour n'importe quel fichier, ouvrez le menu contextuel sur la ligne du fichier (clic droit, ou touche Menu/Applications, ou Maj+F10, sur la ligne sélectionnée).

  • Gérer les pistes... apparaît pour une sortie vidéo MP4 ou MKV.
  • Choisir la piste audio... apparaît pour une extraction audio depuis une vidéo qui contient plusieurs pistes audio.
  • Modifier les métadonnées... apparaît pour les fichiers audio et vidéo. Modifiez les tags standard (titre, artiste, album, etc.) et la pochette, lors de la conversion ou en re-taguant l'original. Pour une vidéo, vous pouvez choisir un type de contenu (Film / Série TV / Autre), et la saison/l'épisode (ou le numéro de piste) peuvent être détectés automatiquement depuis le nom du fichier.

Vous pouvez fusionner deux fichiers ou plus en une seule sortie ; quand la cible est M4B, chaque fichier fusionné devient un chapitre.

Les réglages globaux comme le dossier de sortie, le parallélisme ou les mises à jour se trouvent dans Édition > Préférences.

Prendre en main Accessible Media Converter

Utilisez cette page si vous découvrez l'application. Elle couvre l'ouverture, l'ajout de médias, les formats acceptés et la logique générale des deux onglets.

Démarrage rapide

  1. Ouvrez l'application.
  2. Ajoutez un ou plusieurs fichiers médias.
  3. Choisissez le format de sortie dans la liste de l'onglet actif.
  4. Ouvrez les réglages si vous voulez ajuster la qualité.
  5. Lancez la conversion avec Démarrer la conversion.
L'onglet actif décide du type de conversion : audio vers audio dans l'onglet Audio, vidéo vers vidéo ou vidéo vers audio dans l'onglet Vidéo.

Ajouter des fichiers ou un dossier

Trois méthodes principales existent :

  • Ctrl+O ou Fichier > Ajouter des fichiers...
  • Ctrl+V pour coller des fichiers copiés depuis l'explorateur Windows
  • Fichier > Ajouter un dossier... pour importer un dossier entier

Le collage et l'import de dossier appliquent le même filtrage que l'ouverture classique. Les fichiers non média sont ignorés.

À ce stade, l'application ne propose pas de glisser-déposer. Utilisez l'ouverture, l'import de dossier ou le collage Ctrl+V.

Formats pris en charge

Type Extensions principales
Audio .mp3, .wav, .flac, .aac, .ogg, .wma, .m4a
Vidéo .mp4, .mkv, .avi, .mov, .wmv, .webm
Un fichier texte, un document bureautique ou tout autre format non listé ne doit pas être importé comme média.

Comprendre l'interface

  • Onglet Audio Utilisez-le pour convertir un fichier audio vers un autre format audio.
  • Onglet Vidéo Utilisez-le pour convertir une vidéo vers MP4 ou MKV, ou pour extraire son audio.
  • Liste centrale Chaque ligne affiche le nom du fichier, quelques détails et le statut de traitement.
  • Barre de statut et progression Elles montrent l'état global du batch en cours.

Où trouver les actions avancées

Certaines actions n'apparaissent pas comme de gros boutons dans la fenêtre principale. Pour un fichier vidéo, faites un clic droit sur la ligne du fichier pour ouvrir le menu contextuel.

  • Manage Tracks... apparaît pour une sortie vidéo MP4 ou MKV.
  • Choose Audio Track... apparaît pour une extraction audio depuis une vidéo qui contient plusieurs pistes audio.

Les réglages globaux comme le dossier de sortie, le parallélisme ou les mises à jour se trouvent dans Édition > Préférences.

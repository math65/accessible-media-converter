User guide

Convert videos

For video outputs, the application clearly separates the container, video settings, audio settings, and track management. Keep that structure in mind while working.

On this page

  • Choose a container
  • Adjust video settings
  • Adjust audio settings
  • Manage tracks
  • Edit video metadata

Choose a container

The main video container outputs are MP4 and MKV. You can also convert a video to an audio-only format.

  • MP4 A widely compatible format. Useful audio codec choices are intentionally limited to AAC and MP3.
  • MKV A more flexible format. Useful audio codec choices include AAC, MP3, Opus, and FLAC.

Adjust video settings

The video section first offers a preset, then advanced settings if you want more control.

Video Preset and Encoder Preset do not mean the same thing. The Video Preset is a global profile that sets quality, encoder preset, H.264 profile, and pixel format together. The Encoder Preset only affects encoding speed and compression efficiency for the x264 engine.
Preset Typical use
Compatible Safe choice for simple and robust playback.
Balanced Balance between quality, speed, and file size.
High Quality Higher quality, slower encoding.
Small File Pushes file size lower.
Fast Encode Reduces processing time.

Advanced video settings include:

  • CRF
  • x264 preset
  • H.264 profile
  • pixel format

In practice, if you want a simple setup, start with a Video Preset. If you then change the Encoder Preset, CRF, profile, or pixel format, you move away from the global preset and into a more customized setup.

Adjust audio in a video output

The audio section stays separate from the video section. Choose the audio codec first, then set bitrate, quality, channels, and normalization if needed.

For a video output, the logic is always: container first, then video settings, then audio settings.

Manage tracks

To open track actions, right-click (or the Menu/Applications key, or Shift+F10, on the selected row) a video file in the list. The menu depends on the output format that is currently selected.

  • Manage Tracks... Available for an MP4 or MKV output.
  • Choose Audio Track... Available for audio extraction from a video with several audio tracks.

The track manager then allows you to:

  • keep or remove video, audio, and subtitle tracks;
  • reorder tracks;
  • change the title and language;
  • adjust some dispositions such as default or forced.

For video-to-audio conversion, the application uses only one audio track. If several tracks exist, a dedicated selector lets you choose which one to extract.

If no track action appears on right-click, check the selected output format first and confirm that the selected file is actually a video file.

Edit video metadata

Right-click a video file (or several) in the list, then choose Edit Metadata... (right-click, or the Menu/Applications key, or Shift+F10, on the selected row). A dedicated window opens to edit the tags.

A Content type selector adapts the available fields:

  • Movie Title, Year, Genre, Synopsis, Comment.
  • TV series Episode title, Series, Season, Episode, Year, Genre, Synopsis, Comment.
  • Other Title, Year, Genre, Comment.

An Apply to option controls where the changes go:

  • During conversion (output file) The metadata is written to the converted output.
  • Re-tag the original file now (without re-encoding) The original file is re-tagged in place, with no re-encoding.
Automatic season/episode detection. For a single TV series file, the season and episode (and the series name) are pre-filled from the file name when it contains a marker such as S01E02 or 1x02. In batch mode, the Detect the season and episode from each file name checkbox reads the numbers from each file.
The cover art in the metadata window applies to audio outputs (MP3, AAC/M4A, ALAC, FLAC). For a video output, the cover art options are disabled.

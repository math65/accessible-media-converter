Encoding presets

A preset is a saved set of conversion settings that you can reuse in two clicks. Prepare a "Podcast" or "Audiobook" profile once, then apply it whenever you need it instead of adjusting every option by hand.

Open the presets window

You can open the presets manager from three places:

  • The Presets... button, next to the Settings / Quality button.
  • A Presets... button on the start screen, available even with no file loaded — handy to import a preset before adding anything.
  • The file list: right-click (or the Menu key / Shift+F10) and choose Manage Presets...

Presets are grouped by the active tab: audio presets appear on the Audio tab, video presets on the Video tab, and image presets on the Image tab.

What a preset contains

  • Output format For example MP3, AAC, M4B, MP4, JPEG…
  • Encoding settings Quality, bitrate, mono/stereo, and every other option from the Settings / Quality window.
  • Output destination The output folder and the "keep the original subfolder structure" option.
  • Metadata template A set of shared tags (genre, artist, album…) applied to your files. Images have no metadata template.

Save, apply, and manage presets

The presets window lists your presets by name and offers:

  • Save current settings as preset Captures the current format, encoding settings, and output destination under a name you choose.
  • Apply Restores the selected preset in one step. You can also double-click a preset in the list to apply it.
  • Replace with current settings Overwrites the selected preset with your current settings, keeping its name and metadata template.
  • Edit metadata Edits the preset's shared-tag template (not available for images).
  • Rename and Delete Manage the list. Names are case-insensitive.

Apply to selected files only

Applying a preset always sets the output format, encoding settings, and output destination globally — that is what a preset is for. The metadata template behaves differently depending on how you opened the window:

  • From the Presets... button, the metadata is applied to every loaded file.
  • From the file list right-click menu, the metadata is applied only to the selected files.

Import and export

Export saves your presets to a file you can back up or copy to another computer; Import reads them back in. On import, if a name already exists you are asked whether to overwrite, keep both, or skip.

When you export, you can choose what to include. The output folder and the metadata template can be left out — the output folder in particular rarely makes sense from one computer to another. The format and encoding settings are always included.

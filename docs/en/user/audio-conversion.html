User guide

Convert audio files

The Audio tab is designed to stay simple: choose an output format, then adjust quality, bitrate, channels, and normalization if needed.

Choose an output format

Common audio formats include:

  • MP3
  • AAC / M4A
  • WAV
  • FLAC
  • ALAC / M4A
  • OGG
  • WMA
  • M4B (audiobook with chapters)

The selected format determines which settings are available afterwards. A lossless format such as FLAC does not follow the same logic as a lossy format such as MP3 or AAC.

M4B audiobook (chapters)

The M4B output format produces an audiobook with chapters. M4B is also accepted as input.

  • Merging several files into one M4B creates one chapter per file.
  • Converting a single file that is already chaptered preserves its chapters.
  • Defaults to stereo at 128 kb/s; fully adjustable (for example mono / 64 kb/s for spoken voice).

To assemble several files into one chaptered audiobook, use the merge (select at least two files) with an M4B output. Chapter naming is configured in the preferences.

Adjust quality

  • Copy Stream Copies the existing audio stream without re-encoding when the format allows it.
  • Convert Re-encodes the audio with the selected parameters.
  • CBR Uses a constant bitrate.
  • VBR Uses variable quality to optimize the balance between quality and size.
  • Sample rate, channels, bit depth Let you refine the result depending on the selected codec.
  • Bit depth and FLAC compression These are specific to lossless formats (such as WAV, FLAC, ALAC) and do not apply to lossy formats like MP3 or AAC.

Edit audio metadata

Right-click one or several audio files (or the Menu/Applications key, or Shift+F10, on the selected row), then Edit Metadata.... Edit the tags: title, artist, album, album artist, composer, year, track, disc, genre, comment, as well as Grouping, Copyright and Lyrics (multiline). The cover art can be replaced (JPEG/PNG), removed, or kept.

A Detect track number from the file name checkbox fills in the track number. Choose the target: apply during conversion or re-tag the original now. Batch editing applies the filled-in fields to all selected files.

Streaming normalization

An option can normalize audio for streaming around -16 LUFS. It only applies when the audio is re-encoded.

If you keep the audio in Copy Stream, no normalization is applied.

Batch processing

The application can run several conversions in parallel on the active tab. The maximum number of simultaneous tasks is configured in the preferences.

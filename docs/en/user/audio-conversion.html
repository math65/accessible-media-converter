User guide

Convert audio files

The Audio tab is designed to stay simple: choose an output format, then adjust quality, bitrate, channels, and normalization if needed.

On this page

  • Choose an output format
  • Adjust quality
  • Streaming normalization
  • Batch processing

Choose an output format

Common audio formats include:

  • MP3
  • AAC / M4A
  • WAV
  • FLAC
  • ALAC / M4A
  • OGG
  • WMA

The selected format determines which settings are available afterwards. A lossless format such as FLAC does not follow the same logic as a lossy format such as MP3 or AAC.

Adjust quality

  • Copy Stream Copies the existing audio stream without re-encoding when the format allows it.
  • Convert Re-encodes the audio with the selected parameters.
  • CBR Uses a constant bitrate.
  • VBR Uses variable quality to optimize the balance between quality and size.
  • Sample rate, channels, bit depth Let you refine the result depending on the selected codec.

Streaming normalization

An option can normalize audio for streaming around -16 LUFS. It only applies when the audio is re-encoded.

If you keep the audio in Copy Stream, no normalization is applied.

Batch processing

The application can run several conversions in parallel on the active tab. The maximum number of simultaneous tasks is configured in the preferences.

User guide

Convert images

The Image tab converts an image to another format, with optional resizing. Images added to the application are automatically routed to this tab.

On this page

  • Choose an output format
  • Adjust quality
  • Resize
  • Batch processing

Choose an output format

The supported image formats are:

  • JPEG
  • PNG (lossless)
  • WebP
  • TIFF (lossless)
  • BMP (uncompressed)

The selected format determines which settings are available: a lossy format such as JPEG offers a quality value, while a lossless format such as PNG offers a compression level.

Adjust quality

  • Quality (JPEG, WebP) From 1 to 100. Higher means better quality and a larger file.
  • Compression (PNG) From 0 (fast) to 9 (smallest file), with no loss of quality.
  • Lossless (WebP) A checkbox enables lossless WebP mode.
  • Compression (TIFF) LZW, Deflate, PackBits, or none.

Resize

A resize setting can scale the image down to a target resolution (for example 4K, Full HD, HD). The aspect ratio is preserved automatically, so the image is never distorted.

Batch processing

As with audio and video, the application can convert several images in parallel. The maximum number of simultaneous tasks is configured in the preferences.

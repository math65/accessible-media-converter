Getting started with Accessible Media Converter

Use this page if you are discovering the application. It covers launching, adding media, supported formats, and the general logic of the two tabs.

On this page

  • Quick start

  • Supported formats

Quick start

  1. Open the application.
  2. Add one or more media files.
  3. Choose the output format in the list for the active tab.
  4. Open the settings if you want to adjust quality.
  5. Start the conversion with Start Conversion.
The active tab decides the conversion type: audio to audio in the Audio tab, video to video or video to audio in the Video tab.

Add files or a folder

Three main methods are available:

  • Ctrl+O or File > Add Files...
  • Ctrl+V to paste files copied from Windows Explorer
  • File > Add Folder... to import a whole folder

Paste and folder import use the same filtering as the standard open dialog. Non-media files are ignored.

At this stage, the application does not provide drag and drop. Use file open, folder import, or Ctrl+V paste instead.

Supported formats

Type Main extensions
Audio .mp3, .wav, .flac, .aac, .ogg, .wma, .m4a
Video .mp4, .mkv, .avi, .mov, .wmv, .webm
A text file, office document, or any other format not listed here should not be imported as media.

Understand the interface

  • Audio tab Use it to convert an audio file to another audio format.
  • Video tab Use it to convert a video to MP4 or MKV, or to extract its audio.
  • Main list Each row shows the file name, a few details, and the processing status.
  • Status bar and progress They show the overall state of the current batch.

Where to find advanced actions

Some actions do not appear as large buttons in the main window. For a video file, right-click the file row to open the context menu.

  • Manage Tracks... appears for an MP4 or MKV video output.
  • Choose Audio Track... appears for audio extraction from a video that contains several audio tracks.

Global settings such as the output folder, parallelism, or update checks are available in Edit > Preferences.

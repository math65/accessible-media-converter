- **Choisir de recevoir les pré-versions (nouveau).** Une nouvelle option dans les Préférences, **« Proposer aussi les pré-versions (bêta / versions candidates) »**, vous permet de recevoir les versions de test en avance, en plus des versions stables. Désactivée par défaut. Lorsqu'une version stable est ensuite publiée, elle vous est proposée comme mise à jour : vous ne restez jamais bloqué sur une version de test.
- **Gérer les pistes de plusieurs vidéos à la fois.** Les actions **« Gérer les pistes »** et **« Choisir la piste audio »** tiennent désormais compte de votre sélection. Sélectionnez plusieurs vidéos, faites un clic droit, et le menu affiche **« (N fichiers) »** : vos choix, faits une seule fois, s'appliquent à tous les fichiers sélectionnés, par position de piste — idéal pour une série d'épisodes au même agencement (la 1re piste française, la 2e, etc., sont bien distinguées). Les fichiers ayant moins de pistes sont laissés inchangés et signalés.
- **Options de pistes simplifiées.** Les attributs de piste proposés ont été revus pour ne garder que ceux réellement utiles à chaque type : la vidéo ne montre plus que « Par défaut » ; l'audio conserve « Audiodescription » (l'option « Descriptions », sans rapport avec l'audio, a été retirée) ; les sous-titres gardent « Par défaut », « Forcé » et « Sourds et malentendants ».
- **FFmpeg mis à jour** vers la version 8.1.2.

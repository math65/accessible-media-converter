- **Keep your original subfolder structure (optional).** When you add a folder that contains subfolders and convert into a specific output folder, the app can now recreate the same subfolder tree at the destination instead of dropping every file together. It's a new option in Preferences, **off by default** — turn it on if you want the layout preserved. It has no effect when output goes next to the source file (which already keeps the structure).
- **Two FFmpeg fixes.**
  - Merging files whose names contain an apostrophe (e.g. `Don't Stop.flac`, `O'Brien.mp3`) now works reliably.
  - The "None" (uncompressed) option for TIFF images no longer fails — uncompressed TIFF export works correctly.
- **Updated media engine.** Embedded FFmpeg refreshed to the 2026-06-15 build.

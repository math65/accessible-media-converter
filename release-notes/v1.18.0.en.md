- **Choose whether to get pre-release versions (new).** A new Preferences option, **"Also offer pre-release versions (beta/release candidates)"**, lets you receive test versions early, in addition to stable releases. Off by default. When a stable version is later published, it is offered to you as an update too, so you never get stuck on a test build.
- **Manage tracks across several videos at once.** The **"Manage Tracks"** and **"Choose Audio Track"** actions now follow your selection. Select several videos, right-click, and the menu reads **"(N files)"**: choices made once apply to every selected file, by track position — ideal for a series of episodes with the same layout (the 1st French track, the 2nd, etc. are told apart). Files with fewer tracks are left untouched and reported.
- **Simpler track options.** The track attributes on offer were reviewed to keep only the ones that actually matter per track type: video now shows only "Default"; audio keeps "Audio Description" (the unrelated "Descriptions" option was removed); subtitles keep "Default", "Forced" and "Hearing Impaired".
- **Updated FFmpeg** to version 8.1.2.

- **Encoding presets (new).** Save your encoding settings as named presets and reuse them in two clicks. A new **"Presets…"** button sits next to **Settings / Quality** and opens a dedicated manager:
  - **Save** the current settings as a preset, **Apply** the selected one, **Replace** it with the current settings, **Rename**, **Delete**.
  - **Import** and **Export** presets — handy to back them up or move them to another computer.
  - A preset remembers the output **format**, the **encoding settings** (quality, bitrate, mono/stereo…), the **destination** (output folder + the "keep subfolder structure" option) and a shared **metadata** template (genre, artist, album…).
  - On **export**, you can choose what to include — for example leave out the output folder, which rarely makes sense on another computer.
- **This is a pre-release (rc1)** for testing. It is not offered through the in-app updater; install it only if you were given the link.

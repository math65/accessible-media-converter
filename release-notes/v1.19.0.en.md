- **Works hand in hand with DownAccess.** DownAccess downloads media from the web, and Accessible Media Converter converts it — much like Downie and Permute. From a finished download in DownAccess you can now open the file straight into this app for advanced conversion. A new **Help → "Download Media (DownAccess)…"** entry also points you to DownAccess if you don't have it yet.
- **No more lost files during a conversion.** If a file is sent to the app from outside (from DownAccess, or via the Explorer "Convert with…" command) while a conversion is already running, it is no longer dropped: it now waits and is added to the list automatically once the current conversion finishes, with a clear status message.

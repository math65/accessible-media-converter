- New "Edit Metadata..." command: right-click one or several audio/video files in the list to edit all standard tags (title, artist, album, album artist, composer, year, track number, disc number, genre, comment) and the cover art (replace from a JPEG/PNG image, remove, or keep).
- Cover art editing is available for MP3, AAC/M4A, ALAC and FLAC.
- Batch editing: select several files (for example a whole album) and the fields you fill in are applied to all of them; fields left empty are not changed.
- Two ways to apply: write the metadata into the converted output file (the original is never touched), or re-tag the original file in place instantly, without re-encoding.

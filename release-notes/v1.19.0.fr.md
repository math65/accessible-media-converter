- **Fonctionne main dans la main avec DownAccess.** DownAccess télécharge des médias depuis le web, et Access Media Converter les convertit. Depuis un téléchargement terminé dans DownAccess, vous pouvez désormais ouvrir le fichier directement dans cette application pour une conversion avancée. Une nouvelle entrée **Aide → « Télécharger des médias (DownAccess)… »** vous oriente aussi vers DownAccess si vous ne l'avez pas encore.
- **Plus de fichiers perdus pendant une conversion.** Si un fichier est envoyé à l'application depuis l'extérieur (depuis DownAccess, ou via la commande « Convertir avec… » de l'explorateur) alors qu'une conversion est déjà en cours, il n'est plus ignoré : il patiente et s'ajoute automatiquement à la liste à la fin de la conversion en cours, avec un message d'état clair.

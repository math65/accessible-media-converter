- New **M4B (audiobook) output format with chapters**: when merging several audio files, each file becomes a chapter; when converting a file that already has chapters, its chapters are kept. Bitrate, channels and quality remain fully adjustable (stereo 128 kbps by default, since some audiobooks contain music).
- New "M4B chapter naming" preference: the file's embedded title otherwise "Chapter N" (in the app language), or title otherwise file name, or always numbered.
- Metadata editing redesigned for video: a **Content type** selector (Film / TV series / Other) shows the relevant fields — synopsis for a film; series, season and episode for a TV series.
- **Automatic detection from the file name** while editing: season and episode (for example S01E02) for videos, track number for audio.
- New audio fields: **Grouping**, **Copyright** and **Lyrics** (multi-line).
- Accessibility improvements: "Select all" (Ctrl+A) and the keyboard-opened context menu now respect the multiple selection.
- The update installer now verifies the integrity of the downloaded file (size and SHA-256 digest) before running it.

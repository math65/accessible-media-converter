- New **ABR (target average bitrate)** rate mode for MP3, alongside CBR and VBR: you pick an average bitrate (for example 192 kbps) and the encoder varies the bitrate around that average — the best size/quality trade-off when you don't want a fixed constant bitrate.
- New **"Convert with Accessible Media Converter"** entry in the Windows Explorer context menu (Applications key or Shift+F10), on both files and folders: it opens the app directly on the selection (a folder's contents are added automatically).
- Fix: converting a file while preserving its original metadata could produce a truncated (very short) audio file without reporting any error. The cover art is now kept only when it is safe to do so, and an output check triggers a retry or a real error when the file is unexpectedly short.

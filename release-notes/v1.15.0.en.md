- **Many more input formats accepted.** You can now add a much wider range of files and convert them to your usual formats:
  - **Audio**: `.opus` (voice notes, podcasts, YouTube audio), `.mp2`, `.aiff`/`.aif`, `.ac3` and `.eac3` (Dolby, TV/DVD), `.dts`, `.mka` (Matroska audio), `.amr` (phone voice recordings).
  - **Video**: `.mpg`/`.mpeg`, `.vob` (DVD), `.m4v`, `.3gp`/`.3g2`, `.flv`, `.ogv`.
  - **Images**: `.heic`/`.heif` — **iPhone photos** can now be converted to JPEG, PNG and the other image formats.
  Each file is sorted automatically into the right tab; nothing else changes in how you convert.
- **Improved startup notices.** When the app shows a notice at launch, it can now include a button that opens a related link, and a notice and an available-update prompt will no longer appear stacked on top of each other.

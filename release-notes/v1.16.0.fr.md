- **Conservez l'arborescence de vos sous-dossiers (optionnel).** Quand vous ajoutez un dossier contenant des sous-dossiers et que vous convertissez vers un dossier de sortie spécifique, l'application peut désormais recréer la même arborescence de sous-dossiers à destination au lieu de regrouper tous les fichiers ensemble. C'est une nouvelle option dans les Préférences, **désactivée par défaut** — activez-la si vous voulez conserver l'organisation. Elle n'a aucun effet quand la sortie se fait à côté du fichier source (qui conserve déjà la structure).

- **Réglages de sortie par fichier** : vous pouvez désormais convertir un lot avec un format et une qualité différents pour chaque fichier. Clic droit sur un fichier (ou une sélection multiple) → **« Réglages de sortie… »**, choisissez le format puis ajustez les réglages de qualité habituels ; le choix s'affiche dans la colonne Statut et ne s'applique qu'à ces fichiers. **« Réinitialiser les réglages de sortie »** revient au format global.
- **Réordonner les fichiers de la liste** : changez l'ordre avec **Alt+Haut / Alt+Bas** sur le fichier ciblé, ou via **« Monter » / « Descendre »** dans le menu contextuel. Cet ordre est aussi celui utilisé pour la fusion.
- **Valider les boîtes de dialogue avec Entrée** : les boîtes Réglages, Support, Mise à jour et rapport d'erreur se valident maintenant avec Entrée et s'annulent avec Échap — plus besoin de tabuler jusqu'au bouton Valider. Les champs multilignes (paroles, message) gardent Entrée comme saut de ligne.
- **Correctif — multi-sélection de l'explorateur** : sélectionner plusieurs fichiers dans l'explorateur Windows et choisir « Convertir avec Accessible Media Converter » n'ouvre plus une fenêtre par fichier. Une seule fenêtre reçoit désormais tous les fichiers et revient au premier plan.

- **Per-file output settings**: you can now convert a batch with a different format and quality for each file. Right-click a file (or a multi-selection) → **"Output Settings…"**, pick the format then adjust the usual quality controls; the choice is shown in the Status column and applied only to those files. Use **"Reset Output Settings"** to go back to the global format.
- **Reorder files in the list**: change the order with **Alt+Up / Alt+Down** on the focused file, or via **"Move Up" / "Move Down"** in the context menu. The screen reader announces the moved file and its new position. This also sets the order used when merging files.
- **Press Enter to confirm dialogs**: the Settings, Support, Update and error-report dialogs now validate with Enter and cancel with Escape — no more tabbing all the way to the OK button. Multi-line fields (lyrics, message) keep Enter as a line break.
- **Fix — Explorer multi-selection**: selecting several files in Windows Explorer and choosing "Convert with Accessible Media Converter" no longer opens one window per file. A single window now receives all the files and comes back to the front.

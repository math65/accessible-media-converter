- Added a "View on GitHub" entry in the Help menu that opens the project repository in your default browser.
- Merge errors now include FFmpeg's diagnostic output to make troubleshooting easier.
- Importing a file that ffprobe cannot parse no longer freezes the application (a 30-second timeout has been added).
- Fixed a crash that could occur when a second conversion failed after the first error report dialog had been dismissed.
- Conversion errors caused by a missing video track now include the affected filename.
